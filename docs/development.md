# Development

1. Clone the repository:

   ```shell
   git clone https://github.com/aeisenbarth/myst-nb-json.git
   ```

2. Install dependencies:

   ```shell
   pip install pdm
   pdm install
   ```

3. Run tests:

   ```shell
   pdm test
   ```

4. Build documentation:

   ```shell
   pdm docs
   ```
